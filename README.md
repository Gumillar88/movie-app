# Movie App

## Arsitektur Proyek

- **Frontend:** React.js
- **Backend:** Node.js dengan Express
- **API:** OMDb API

## Pustaka yang Digunakan

- axios
- react-router-dom
- express
- cors
- jsonwebtoken
